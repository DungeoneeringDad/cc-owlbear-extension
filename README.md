# Castles & Crusades Record Sheet — Official-Style Pass

This version restyles the Owlbear Rodeo extension to feel more like a traditional tabletop record sheet:
- parchment paper palette
- serif typography
- boxed record-sheet sections
- more traditional C&C field naming

## Publish
1. Upload all files to the root of a GitHub repository.
2. Enable GitHub Pages.
3. Install in Owlbear Rodeo using:

https://YOUR-USERNAME.github.io/YOUR-REPO/manifest.json
